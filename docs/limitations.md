# Limitations

1. Actual NIL payments are private and incomplete. This project uses public NIL valuation as a proxy, not confirmed NIL spending.
2. On3 public pages may only show a limited set of players, especially without a subscription.
3. NIL valuation can reflect marketability and social following, not only football ability.
4. Team performance is affected by many factors that are difficult to fully control for, including injuries, coaching changes, schedule difficulty, transfers, and player development.
5. Athletics finance data may be reported by fiscal year, while football performance is reported by season. The project treats them as close proxies but should acknowledge timing differences.
6. Public finance data often captures athletics department spending, not donor collective spending directly.
7. The model identifies relationships, not guaranteed causation.
