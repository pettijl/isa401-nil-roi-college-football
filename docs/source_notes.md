# Source Notes

## On3 NIL Valuations

Used as a public proxy for NIL/player market value. The project does not treat these as verified payments. On3's NIL Deal Tracker states that financial data is undisclosed and private to the athlete, so the analysis should describe On3 valuation as a proxy.

Primary page:

- https://www.on3.com/nil/rankings/player/college/football/

Position pages can also be scraped:

- `?position=qb`
- `?position=rb`
- `?position=wr`
- `?position=te`
- `?position=ot`
- `?position=iol`
- `?position=edge`
- `?position=dl`
- `?position=lb`
- `?position=cb`
- `?position=s`
- `?position=ath`
- `?position=k`
- `?position=p`
- `?position=ls`

## CollegeFootballData / cfbfastR

Used for performance, team records, team talent, recruiting, SRS, and team stats.

Main package:

- https://cfbfastR.sportsdataverse.org/

API key:

- https://collegefootballdata.com/key

## Knight-Newhouse / EADA Finance Data

Used for broader athletics finance and football spending proxies.

Knight-Newhouse database:

- https://knightnewhousedata.org/

EADA database:

- https://ope.ed.gov/athletics/

Recommended finance variables:

- total athletic revenue
- total athletic expenses
- football spending
- football coaching salaries
- donor contributions, if available
