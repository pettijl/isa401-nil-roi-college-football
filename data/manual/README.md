# Manual Data Folder

This folder is for public finance downloads or manual correction files.

## on3_school_crosswalk.csv

The On3 scraper may not perfectly capture school names because teams are often stored as image alt text. This file lets you manually correct school names by player.

## athletics_finance_manual.csv

Replace this file with a download from Knight-Newhouse or EADA, or paste the finance variables into this template.

Required columns:

- season
- school

Recommended columns:

- total_athletic_revenue
- total_athletic_expenses
- football_spending
- football_coaching_salaries
- donor_contributions
- source
